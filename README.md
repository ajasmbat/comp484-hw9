## Reflection

The easiest part was setting up the HTML file and linking the external JavaScript file since we had a starter template to follow. The hardest part was remembering that JavaScript months start at 0 instead of 1, which made formatting the date tricky at first. I learned that the Date object requires you to manually adjust the month by adding 1 to get the correct value. I also learned that Number() can convert strings to numbers but returns NaN when the string is not a valid number, and Number.isNaN() is useful for checking that. Displaying results in the browser using getElementById and innerHTML was straightforward once I understood how to build strings with HTML tags like br for line breaks.
